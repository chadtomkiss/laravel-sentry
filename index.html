Laravel-Sentry

A simple app to get started with Laravel 4 and Cartalyst Sentry

Download this project as a .zip file Download this project as a tar.gz file

Hey m8s. Just a simple Laravel + Sentry app to save you time writing signup and login.

Getting Started

Create a local database config

/app/config/local/database.php

Run Migrations

php artisan migrate --package=cartalyst/sentry --env=local
php artisan migrate --env=local

Routes

{{ URL::route('account.signup') }}
{{ URL::route('account.login') }}

Notes

$current_user is shared to all Views by using the App::before function in filters.php

This is the bare minimum to get started, nothing fancy like implementing the Repository pattern etc.
